
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Create a classic Hangman game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Develop a fully functional Hangman game that randomly selects a word from a predefined list and allows players to guess letters one at a time. The game should track progress, display the current state of the word with correct guesses revealed, and determine win/lose conditions.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (e.g., `_ _ _ _` format)
- Track incorrect guesses remaining and display them
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages with game summary
